# Affiliate Flow Studio

Affiliate Flow Studio helps affiliate marketers decide which products are worth promoting and gives ready-to-use content angles.

## V1 Scope

- Product input form
- Product scoring engine
- Result dashboard
- Hook/script/caption template generator
- Product history
- Local storage first
- Supabase later

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- LocalStorage for V1
- Supabase optional in V2
- AI API optional in V2
