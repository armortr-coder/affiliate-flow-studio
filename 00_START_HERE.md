# Affiliate Flow Studio - Codex Only Plus Pack

This pack is designed for Arm who currently has only Codex Plus.

Important decision:
- Do NOT start with a complex AI automation system.
- Build a usable MVP first.
- V1 works without Claude API or OpenAI API.
- AI integration is optional in V2 after the product works.

MVP Core:
Paste or enter product data -> calculate product score -> show recommendation -> generate content templates.

Why this direction:
Codex Plus can help you code, but your web app cannot call AI automatically unless you also use an API key.
So V1 must not depend on external AI APIs.
