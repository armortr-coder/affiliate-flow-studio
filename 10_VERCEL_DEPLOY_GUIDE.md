# Vercel Deploy Guide

## 1. Push to GitHub

```bash
git init
git add .
git commit -m "initial affiliate flow studio mvp"
```

Create a GitHub repository and push.

## 2. Deploy on Vercel

- Go to Vercel
- Import GitHub repository
- Framework: Next.js
- Deploy

## 3. After deploy

Check:
- Mobile layout
- Product form
- Score result
- History save
